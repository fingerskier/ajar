Martial Arts Program


6 Keys
- Athleticism
- Power
- Mobility
- Technique
- Combat
- Rest

Concentrate on one of these keys each day.
Perhaps do a secondary key after the first.


e.g.

Athleticism
===========
Skipping
Lunges
Wheelbarrows
Lizarding
Bear crawl
Beast in&outs
Crabwalks


Power
=====
Plyo push-up
One-leg squat
Nordic curl
Pull-up


Mobility
========
Shoulder rolls
Shrimping
Clamshells
Table-tops


Technique
=========
Focus pad kicks & punches
Kata/forms


Combat
======
Free sparring
Sparring drills


Rest
====
Visualization
