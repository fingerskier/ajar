# Guitar Setup

* Neck
  * Truss Rod
    * tighten for less bow/relief
    * loosen for more bow/relief
  * Nut
    * teflon/silicon lubricant
  * Fretboard
    * Clean and oil
  * Frets
    * Level
    * Crown
    * Polish
      * 1000 grit sandpaper
* Bridge
* Pickups
  * Height
    * higher for more output and more compression
    * lower for lower output and more dynamics
